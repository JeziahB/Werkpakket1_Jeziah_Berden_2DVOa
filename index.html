CPU

The Rock Assistant runs on the Radxa Rock 5B Single Board Computer using an 8 core Rockchip RK3588 SoC. This is a combined ARM quad core Cortex - A76 CPU @ 2.2/2.4GHz and a quad core Cortex - A55 CPU @ 1.8GHz and an ARM Mali - G610MC4 GPU.

Memory

The Rock Assistant has 8 GB of 64bit DDR4 memory.

Storage

A 256 or 512 GB Samsung NVME SSD is the Boot, OS, Applications & Data drive.
